# Data Type

```
void main() {
  int intNum = 1;
  print(intNum);

  double doubleNum = 2.2;
  print(doubleNum);

  bool isTrue = true;
  print(isTrue);

  String str1 = 'str1';
  print('$str1 - ${str1.runtimeType}');
  
  
  // nullable, non-nullable
  String? str2 = null
  String str3 = 'non-nullable'
}
```

```
// console
1
2.2
true
str1 - String
```
